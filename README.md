##STARTUP


#Create venv

```python -m venv venv``

#install requirements

```pip install```

#move into ultimate_texas_holdem file

```cd ultimate_texas_holdem```

#create superuser

```python manage.py createsuperuser  ```


#run migrations

```python manage.py makemigrations```

```python manage.py migrate```


#run app

```python manage.py runserver```
